# Components (詳細ドキュメント)

このディレクトリには各コンポーネントの使用詳細と図を格納します。

## ダイアグラム

- `class_diagram.puml` — プロジェクト全体のクラス図（PlantUML）
- `sequence_diagram.puml` — 代表的なフロー（MQTT→処理→ストレージ、Web カメラプレビュー）のシーケンス図（PlantUML）

PlantUML ファイルはそのまま PlantUML ツールでレンダリングできます。
